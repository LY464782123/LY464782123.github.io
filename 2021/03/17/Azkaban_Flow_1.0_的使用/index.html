发表于 2021-03-17 | 分类于 notes ， Azkaban | | 阅读数 次

一、简介

Azkaban 主要通过界面上传配置文件来进行任务的调度。它有两个重要的概念：

  • Job： 你需要执行的调度任务；
  • Flow：一个获取多个 Job 及它们之间的依赖关系所组成的图表叫做 Flow。

目前 Azkaban 3.x 同时支持 Flow 1.0 和 Flow 2.0，本文主要讲解 Flow 1.0 的使用，下一篇文章会讲解 Flow 2.0 的使用。
